# Creative AI Homepage

A sleek, responsive website showcasing a futuristic AI platform with a dynamic hero section, 3D and video backgrounds, and smooth animations. Built with HTML, CSS, and JavaScript, it delivers a seamless user experience across desktop and mobile devices using Bootstrap, Vanta.js, Particles.js, and Three.js.

---

## Setup Instructions

1. **Clone the Repository:**

   ```bash
   git clone https://github.com/jaideepj2004/Creative_AI_Homepage.git
   ```

2. **Navigate to the Folder:**

   ```bash
   cd Creative_AI_Homepage
   ```

3. **Run the Website:**

   * Open `index.html` in your preferred web browser (Chrome recommended).

---

## Technologies Used

* **HTML5**: Page structure and semantics
* **CSS3**: Responsive styling and animations
* **JavaScript**: Interactivity for cursor sparkles, scroll reveals, and background media swapping
* **Bootstrap 5.3**: Grid system and utility classes
* **Vanta.js**: Cycling 3D background effects (`NET`, `BIRDS`, `WAVES`)
* **Particles.js**: Particle animations on hover
* **Three.js**: 3D model rendering and cursor-driven rotation
